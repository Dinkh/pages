Welcome to the neo.mjs online examples!

The online demos are not optimised for mobile yet, please use a desktop browser!

There are 3 different versions (dist/production, dist/development, development mode).

The most impressive one is the development mode at the bottom, which runs without any Javascript related builds.

dist/production

Webpack based builds without source-maps and minified files. These versions work in Chrome, Firefox & Safari.

  • COVID-19 Dashboard App (new)
  • Realworld App
  • COVID-19 Gallery
  • COVID-19 Helix
  • Realworld v2 App (in progress, only gallery, helix & home work)
  • Docs App (including the Real World App source documentation)
  • component.DateSelector
  • component.Gallery
  • component.Helix
  • form.field.Date
  • form.field.Select
  • tab.Container

dist/development

Webpack based builds using source-maps and non minified files. These versions work in Chrome, Firefox & Safari.

  • COVID-19 Dashboard App (new)
  • Realworld App
  • COVID-19 Gallery
  • COVID-19 Helix
  • Realworld v2 App (in progress, only gallery, helix & home work)
  • Docs App (including the Real World App source documentation)
  • component.DateSelector
  • component.Gallery
  • component.Helix
  • form.field.Date
  • form.field.Select
  • tab.Container
