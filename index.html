Welcome to the neo.mjs online examples!

For the best user experience, you need to use Chrome with the following browser flag:

chrome://flags/#enable-experimental-web-platform-features
(copy the link into a new Chrome tab)

With this flag, it is possible to get the real JS module code into your browser, no builds required.

  • Realworld App
  • neo.mjs: Docs App (including the Real World App)
  • Helix Demo
  • Gallery Demo

Without the Chrome flag (or for other browsers like Firefox & Safari, you can use the webpack build versions:

  • Realworld App (dist/development)
  • Realworld App (dist/production)
